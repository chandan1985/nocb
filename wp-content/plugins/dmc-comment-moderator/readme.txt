Name: DMC Comment Moderator
Version: 1.0
Author: Pallavi Unkule

This plugin is used to change the email address to whom the comment notification will be sent (instead of the admin email address).

Installation
============
1. Upload the audio-player folder to your '/wp-content/plugins' folder
2. Activate 2 plugins through the Plugins section in your WordPress admin, namely; DMC Comment Moderator, DMC Comment Notification
