== Description ==

*Custom Byline* is a very light plugin that lets you easily and safely add custom byline in normal posts. It doesn’t literally rename or change files in core, nor does it add rewrite rules. 


== Installation ==

1. Go to Plugins › Add New.
2. Search for *Custom Byline*.
3. Look for this plugin, download and activate it.
4. The page will redirect you to the settings. Change your login url there.
