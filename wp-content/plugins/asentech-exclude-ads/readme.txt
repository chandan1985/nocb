Interstitial Ads

== Description ==

Interstitial Ads is an easy to use ads manager for a full page ad that appears before the actual webpage. Ads that appear in a separate page and returns back to previous page after 15 seconds, its implemented as per google standards.


Features:

* Full page ads that gets noticed
* Implemented as per google dfps standards



== Installation ==

1. Upload the `asentech-interstitial-ads` folder to the `/wp-content/plugins/` directory
2. Activate the asentech-Interstitial-Ads plugin through the 'Plugins' menu in WordPress
3. Activate the plugin at the plugin administration page.
4. Create a page as welcome-ad which will be used for inserting Text to display on right hand side of ad .
